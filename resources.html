Resources

Here is a compilation of resources that we've found useful or think are useful for learning data science and coding skills necessary for our projects. Most are free and online accessible, but there are cases where a resource is only available for purchase. Enjoy!

Python

    Beginner

    • Practice Python (website): Thaddaeus' favorite site to recommend for people starting out with coding.
    • Programming for Everybody: Getting Started with Python (Coursera): First course in Michigan State's Python for Everybody specialization series on Coursera.
    • Whirlwind Tour of Python (book; free online version): "a fast-paced intro to Python aimed at researchers and engineers"

    Intermediate

    • Python Data Science Handbook (book; free online version): "a comprehensive introduction to the most important data science tools in the Python"
    • Python Data Structures, Introduction to Data Science in Python, and Applied Plotting, Charting & Data Representation in Python (Coursera): three more of the courses in the University of Michigan's Python for Everybody specialization series on Coursera, a staple recommendation by Owen Lester.

    Advanced

    • Deep Learning with Python (book): this is the book Thaddaeus used to get started with Keras and Deep Learning
